This JavaScript automation script is designed to bulk delete chat history items from the Perplexity AI library or history page directly from the browser console.

The script continuously scans the webpage for selectable chat items, automatically selects them using checkboxes, triggers the delete action, confirms the deletion popup, and repeats the process until no more items remain.

It also includes auto-scrolling functionality to dynamically load additional history entries when the page uses infinite scrolling or lazy loading.
